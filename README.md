## 欢迎贡献

由于游戏厂商众多等原因，欢迎各位玩家提交 PR 或者直接通过 Telegram 联系 https://t.me/BlindGamer 发送。
按照 Telegram 的要求，所提交的文件必须符合以下要求：
1. 文件名为相关厂商名称，后缀格式为 png / webp 两种均可。
2. 图像背景为透明。
3. 图像分辨率为 100 x 100。
